# **App Name**: BizkaiaVet Web

## Core Features:

- Botón de Urgencias: Botón visible para contacto inmediato en caso de emergencias, redirigiendo a un método de contacto rápido.
- Sistema de Citas Online: Flujo de reserva de citas en tres pasos: Servicio -> Veterinario/Hora -> Confirmación. Procesamiento inmediato y confirmación por SMS.
- Perfiles de Veterinarios: Perfiles que incluyen mascotas personales y pasiones fuera de la clínica para generar confianza.
- Páginas de Servicios: Páginas dedicadas a servicios (ej., cirugía, dermatología) diseñadas para SEO y la educación del dueño de la mascota.
- Widget de Reseñas de Google: Widget integrado que muestra reseñas de 5 estrellas de Google filtradas para generar confianza y mostrar comentarios positivos.

## Style Guidelines:

- Color primario: Verde cálido y acogedor (#90EE90) para evocar la naturaleza y un ambiente acogedor.
- Color de fondo: Verde claro desaturado (#F0FFF0), para un telón de fondo suave y natural.
- Color de acento: Marrón terroso (#A0522D) para complementar el verde, agregando profundidad y una sensación de confiabilidad.
- Fuente del encabezado: 'Belleza' para una mezcla de personalidad alineada con el arte y el diseño, perfecta para encabezados impactantes. Dado que se puede anticipar un texto largo, 'Alegreya' servirá para el cuerpo del texto, ya que combina muy bien con 'Belleza'.
- Usar íconos personalizados inspirados en la naturaleza. Agregue una sensación familiar que represente los servicios y la atención brindada.
- Diseño mobile-first, diseño limpio, evitando sliders automáticos. Optimizar para velocidad y facilidad de navegación.
- Animaciones sutiles y naturales para guiar las interacciones del usuario sin abrumar ni ralentizar el sitio.
